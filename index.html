Kegiatan Selama PKL

Selama PKL, saya membantu pengelolaan dokumen, seperti menyusun laporan, memindai arsip, dan mengorganisasi data digital.

Saya belajar menggunakan software untuk membantu proses administrasi, seperti Microsoft Excel, Word, dan sistem manajemen kantor.

Kerja Sama Tim

Saya berinteraksi dengan staf dan rekan kerja untuk menyelesaikan tugas bersama, meningkatkan kemampuan komunikasi dan kolaborasi.

Presentasi dan Dokumentasi

Saya membuat dokumentasi kegiatan harian dan menyusun laporan akhir untuk dipresentasikan di depan pembimbing dan rekan kerja.
